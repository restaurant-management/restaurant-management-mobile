# Usage

## Clone

`git clone git@github.com:restaurant-management/restaurant-management-mobile.git`
`cd restaurant-management-mobile`

## Get package

`flutter pub get`

## Add your google-sevices.json to android/app/

## Apply launcher icon

`flutter pub run flutter_launcher_icons:main`
